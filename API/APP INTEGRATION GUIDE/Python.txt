-----------------------------------PYTHON INTEGRATION-----------------------------------


To integrate Sulfur into a python API, drag the Sulfur folder into your root folder. 

Then import sulfur into a script/ your script via:

"import SulfurAI"

OR read the ReadMe.md file for easier integration.

-----------------------------------SCRIPTS:

*For some functions the information is only in the python documentation, to see this use an IDE such as vscode or PyCharm*

SulfurAI.run_locally("str") - runs sulfur locally. Returns a dictionary that you can explore and use or gain via using your own custom method.
